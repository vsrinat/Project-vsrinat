Student Database Management System (MySQL)

Overview:

This project is a simple Student Database Management System built using MySQL. It allows storing, managing, and analyzing student records through various SQL operations.

Key Features:
- Create and manage student records.
- Retrieve and filter students by grade, gender, GPA, etc.
- Update student data.
- Perform aggregations (average GPA).
- Rename table and retrieve top-performing students.

Files:
- `studentDB.sql` – contains all SQL queries
- `screenshots/` – includes output screenshots
- `README.md` – project description
