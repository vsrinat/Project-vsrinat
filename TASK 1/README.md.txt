Academic Management System (MySQL)

Overview:
This project is a simple Academic Management System built using MySQL. It helps manage student, course, and enrollment data through SQL operations.

Key Features:
- Create and manage `StudentInfo`, `CoursesInfo`, and `EnrollmentInfo` tables
- Insert sample records
- Retrieve student and course details
- Perform analytics using JOINs and aggregation

Files:
- `academicDB.sql` – contains all SQL queries
- `screenshots/` – includes output screenshots
- `README.md` – project description
