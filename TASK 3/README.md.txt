Event Management System (MySQL)

Overview:
This project is a simple Event Management System built using MySQL. It manages event details, attendee information, and event registrations.

Key Features:
- Create and manage `Events`, `Attendees`, and `Registrations` tables
- Insert, update, and delete event data
- Register attendees for events
- Perform reporting: attendee lists, registration count, and revenue

Files:
- `Event Management DB.sql` – contains all SQL queries
- `screenshots/` – includes query output screenshots
- `README.md` – project description
